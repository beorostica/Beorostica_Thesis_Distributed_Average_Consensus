
This folder contains the neccesary files to display results of the hadaware implementarion 
for the broadcast gossip and push sum protocols deployed in our local laboratory.

First, extract "Data_Hardware_Consensus_TopologyAll.zip"

Just run the file "run_hardware_results.m".

The folder "Data_Hardware_Consensus_TopologyAll" contains the results of the every beaglebone black
used in the testbed.
